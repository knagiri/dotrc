### チートシート

プラグイン管理 (lazy.nvim)

プラグインのインストール/更新: :Lazy


ファイル検索 (Snacks picker)

スマートファイル検索: <leader><space>
ファイル検索: <leader>ff
Grep検索: <leader>/
バッファ検索: <leader>,
最近のファイル: <leader>fr
Gitファイル検索: <leader>fg
プロジェクト検索: <leader>fp
設定ファイル検索: <leader>fc
ファイルエクスプローラ: <leader>ft


Git 関連

Gitブランチ: <leader>gb
Gitログ: <leader>gl
Gitステータス: <leader>gs
Git Diff: <leader>gd
GitHub Issues: <leader>gi
GitHub PRs: <leader>gp
Diffview 開く: <leader>do
Diffview 閉じる: <leader>dc
ファイル履歴: <leader>dh


LSP 関連

定義へジャンプ: gd
宣言へジャンプ: gD
実装へジャンプ: gi
ホバー情報表示: K
シグネチャヘルプ: <C-k>
リネーム: <space>rn
コードアクション: <space>ca
参照検索: gr
型定義へジャンプ: gy
フォーマット: <space>lf
LSPシンボル: <leader>ss
ワークスペースシンボル: <leader>sS


補完 (blink.cmp)

補完メニュー表示: <C-Space>
補完キャンセル: <C-e>
補完確定: <CR>
ドキュメントスクロール: <C-b> / <C-f>
スニペット移動: <Tab> / <S-Tab>


検索・その他

バッファ内行検索: <leader>sb
バッファGrep検索: <leader>sB
単語検索: <leader>sw
診断情報: <leader>sd
ヘルプ: <leader>sh
キーマップ一覧: <leader>sk
Undo履歴: <leader>su
レジスタ: <leader>s"
コマンド履歴: <leader>:


診断情報

診断情報のフロート表示: <leader>e
診断リストの表示: <leader>E


AI アシスタント (Sidekick)

Sidekick トグル: <C-.> / <leader>aa
Claude トグル: <leader>ac
セッション選択: <leader>as
行/選択を送信: <leader>at
ファイルを送信: <leader>af
ビジュアル選択を送信: <leader>av
プロンプト選択: <leader>ap


カスタム設定

設定ファイルの場所: ~/.config/nvim/init.lua
設定モジュール: ~/.config/nvim/lua/config/ (options, keymaps, autocmds, lazy)
プラグイン設定: ~/.config/nvim/lua/plugins/ (editor/, lsp/, ui/)


注意: <leader> キーはスペースキーに設定されています。
